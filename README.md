✈️ Travel Booking App UI

## 📌 Project Overview

This is a *Travel Booking App UI Design* created using *Figma*. The application provides a simple and attractive interface for users to book travel destinations. It includes destination selection and booking details.

---

## 🎯 Objective

The main objective of this project is to design a user-friendly travel booking application with a clean and modern interface that makes booking easy and convenient.

---

## ✨ Features

- Main Page
- Create Account
- Eiffel Tower Screen
- Taj Mahal screen
- Great Wall Of China Screen
- Hollywood Screen

---

## 🛠️ Tools Used
